# Proyecto de Machine Learning para Forecasting de Ventas

Este proyecto contiene la estructura básica para desarrollar modelos de machine learning enfocados en la predicción de ventas.

## Estructura de Carpetas
- `data/raw/`: Datos originales sin procesar
- `data/processed/`: Datos procesados y listos para análisis/modelado
- `notebooks/`: Jupyter notebooks para exploración y desarrollo
- `models/`: Almacenamiento de modelos entrenados
- `app/`: Aplicación Streamlit para visualización y predicción
- `docs/`: Documentación del proyecto

## Requisitos
Instala las dependencias con:
```bash
pip install -r requirements.txt
```

## Uso
1. Coloca tus datos en `data/raw/`
2. Procesa los datos y guarda en `data/processed/`
3. Desarrolla y prueba modelos en `notebooks/`
4. Guarda modelos en `models/`
5. Ejecuta la app de Streamlit desde la carpeta `app/`

## Para ejecutarla:
```bash
streamlit run app/app.py
